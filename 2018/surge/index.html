Servicio de despliegue de paginas estáticas Surge

Instalación surge

Lo primero que haremos será instalar Node.js.

$ sudo apt install nodejs

Ahora instalaremos surge con npm (si no lo tenemos lo instalamos):

$ sudo apt install npm
$ sudo npm install --global surge

Ejecución

Ahora ejecutaremos surge desde el directorio donde tenemos nuestra pagina, en mi caso será la siguiente:

vagrant@estatica:~/hugo/p2/public$ ls
2018  404.html  css  img  index.html  index.xml  page  post  sitemap.xml

Ejecutamos surge:

vagrant@estatica:~/hugo/p2/public$ surge
Running as correo@hotmail.com (Student)

            project: /home/vagrant/hugo/p2/public/
         domain: arias.surge.sh
         upload: [====================] 100% eta: 0.0s (10 files, 486476 bytes)
            CDN: [====================] 100%
             IP: 45.55.110.124

   Success! - Published to arias.surge.sh

Con esto ya tendriamos nuestra pagina alojada en arias.surge.sh. Si por algún motivo ya no queremos usar el dominio ponemos la siguiente instrucción:

surge teardown arias.surge.sh
